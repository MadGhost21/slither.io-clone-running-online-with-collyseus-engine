# 🎨 UI & Aesthetic Systems

### Theme: Neon Electric Blue
Strand.io uses a high-contrast dark theme with glowing cyan and deep blue accents.

### Components:
1.  **Skin Studio**: Allows custom multi-color snake generation. Skins are stored as `custom_#hex_#hex` strings.
2.  **Global Chat**: A real-time messaging panel with notification alerts.
3.  **Leaderboard**: Dynamic top-10 display, highlighting the player's own position in white.
4.  **Minimap**: A 120x120 live radar showing food clusters and all player positions.
